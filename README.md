# Coach Loan Vũ - Personal Website

Website cá nhân của Coach Vũ Kiều Loan – F&B Startup Coach (ICF PCC).

## Giới thiệu
Hệ sinh thái dịch vụ hỗ trợ chủ quán F&B hướng tới kinh doanh bền vững thông qua 3 chương trình chính:
- **Passion to Profit**: Chuyển đổi đam mê thành lợi nhuận.
- **Business to Freedom**: Từ vận hành đến tự do.
- **Business Mastery**: Coaching 1:1 chiến lược dài hạn.

## Công nghệ sử dụng
- HTML5 / CSS3 (Vanilla CSS)
- JavaScript (Web Components)
- Google Fonts (Inter, Outfit)

## Cấu trúc thư mục
- `/css`: Chứa các tệp style hệ thống.
- `/js`: Chứa logic header/footer và các hiệu ứng.
- `/assets`: Hình ảnh và tài liệu hướng dẫn.

## Cách xem website
Bạn chỉ cần mở tệp `index.html` trên bất kỳ trình duyệt nào.
